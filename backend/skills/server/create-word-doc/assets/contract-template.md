## 第一章 总则

### 第一条 合同目的

为明确双方的权利义务，根据《中华人民共和国合同法》及相关法律法规，甲乙双方本着平等、自愿、公平和诚实信用的原则，就{{ purpose }}事宜签订本合同，以资共同遵守。

### 第二条 定义

除非本合同另有约定，以下术语具有下列含义：

1. **本合同**：指甲乙双方签订的{{ contract_name }}及其附件。
2. **服务/产品**：指本合同约定的{{ service_or_product }}。
3. **工作日**：指中华人民共和国法定节假日以外的日期。

#### 5.1 乙方权利

## 第四章 违约责任

### 第九条 甲方违约

（一）甲方未按约定支付合同金额的，每逾期一日，应按未付款项的{{ penalty_rate }}%向乙方支付违约金；
（二）{{ additional_a_penalty }}。

### 第十条 乙方违约

（一）乙方未按约定提供服务的，每逾期一日，应按合同金额的{{ penalty_rate }}%向甲方支付违约金；
（二）乙方提供的产品/服务不符合约定标准的，应负责免费更换或返工；
（三）{{ additional_b_penalty }}。

### 第十一条 不可抗力

因不可抗力导致本合同无法履行或部分无法履行时，遭遇不可抗力的一方应及时书面通知对方，并在{{ days }}日内提供相关证明文件。双方可协商解除或部分履行本合同。
