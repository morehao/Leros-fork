# 文档模板包

本文件包含常用文档模板的占位符和结构说明。

---

## 1. 决策日志 (Decision Log)

### 用途
记录团队决策及其背景、依据和影响。

### 占位符

| 字段 | 占位符 | 说明 |
|------|--------|------|
| 决策 ID | `{{ decision_id }}` | 唯一标识符，如 DEC-2024-001 |
| 决策标题 | `{{ title }}` | 决策的简明描述 |
| 日期 | `{{ date }}` | 决策日期 |
| 决策者 | `{{ decision_maker }}` | 做出决策的人 |
| 状态 | `{{ status }}` | 提议/已批准/已执行/已废弃 |
| 优先级 | `{{ priority }}` | 高/中/低 |

### 结构模板

```
# 决策日志

## {{ decision_id }}: {{ title }}

**日期**: {{ date }} | **决策者**: {{ decision_maker }}
**状态**: {{ status }} | **优先级**: {{ priority }}

### 问题背景
（描述需要决策的问题）

### 备选方案
| 方案 | 优点 | 缺点 |
|------|------|------|
| 方案 A | | |
| 方案 B | | |

### 决策理由
（说明选择该方案的原因）

### 预期影响
- 影响 1
- 影响 2

### 后续行动
| 行动项 | 负责人 | 截止日期 |
|--------|--------|----------|
| | | |

### 关联文档
- 链接 1
- 链接 2
```

## 2. 会议记录 (Meeting Notes)

### 用途
记录会议讨论要点、决策和行动项。

### 占位符

| 字段 | 占位符 | 说明 |
|------|--------|------|
| 会议主题 | `{{ title }}` | 会议标题 |
| 日期时间 | `{{ datetime }}` | 会议时间 |
| 地点 | `{{ location }}` | 会议地点 |
| 主持人 | `{{ host }}` | 会议主持人 |
| 记录人 | `{{ recorder }}` | 记录人员 |
| 参与者 | `{{ participants }}` | 参会人员列表 |

### 结构模板

```
# {{ title }}

**日期**: {{ datetime }}
**地点**: {{ location }}
**主持人**: {{ host }}
**记录人**: {{ recorder }}

## 参会人员
{{ participants }}

## 议程

3. 议题 3

## 讨论要点

### 议题 1: {{ topic_1 }}
- 要点 1
- 要点 2

- 要点 1
- 要点 2

## 决策
| 决策 | 决策者 | 日期 |
|------|--------|------|
| | | |

## 行动项
| 行动项 | 负责人 | 截止日期 | 状态 |

## 下次会议
- **时间**: {{ next_meeting_datetime }}
```

## 3. 变更日志 (Changelog)

### 用途
记录项目或产品的变更历史。

### 占位符

| 字段 | 占位符 | 说明 |
|------|--------|------|
| 版本号 | `{{ version }}` | 语义化版本，如 v1.2.0 |
| 发布日期 | `{{ release_date }}` | 变更发布日期 |
| 类型 | `{{ change_type }}` | 新功能/修复/改进/破坏性变更 |

### 结构模板

```
# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [{{ version }}] - {{ release_date }}

### Added
- 新功能 1
- 新功能 2

### Changed
- 变更 1
- 变更 2

### Deprecated
- 弃用功能 1（将在下版本移除）

### Removed
- 移除功能 1

### Fixed
- 修复 1
- 修复 2

### Security
- 安全修复 1

---

## [{{ version }}] - {{ release_date }}

（历史版本...）
```

## 4. 问题跟踪 (Issue Tracker)

### 用途
跟踪问题、缺陷和任务。

### 占位符

| 字段 | 占位符 | 说明 |
|------|--------|------|
| 问题 ID | `{{ issue_id }}` | 唯一标识符 |
| 问题标题 | `{{ title }}` | 问题简述 |
| 严重程度 | `{{ severity }}` | 严重/高/中/低 |
| 优先级 | `{{ priority }}` | P0/P1/P2/P3 |
| 状态 | `{{ status }}` | 待处理/进行中/已解决/已关闭 |
| 报告人 | `{{ reporter }}` | 报告人 |
| 负责人 | `{{ assignee }}` | 处理人 |
| 创建日期 | `{{ created_date }}` | 创建时间 |
| 截止日期 | `{{ due_date }}` | 计划完成时间 |

### 结构模板

```
# 问题: {{ issue_id }}

## 基本信息

| 字段 | 值 |
|------|-----|
| 标题 | {{ title }} |
| 严重程度 | {{ severity }} |
| 优先级 | {{ priority }} |
| 状态 | {{ status }} |
| 报告人 | {{ reporter }} |
| 负责人 | {{ assignee }} |
| 创建日期 | {{ created_date }} |
| 截止日期 | {{ due_date }} |

## 问题描述
（详细描述问题）

## 复现步骤
1. 步骤 1
2. 步骤 2
3. 步骤 3

## 预期行为
（描述期望的行为）

## 实际行为
（描述实际发生的行为）

## 截图/日志
![截图或日志]

## 解决方案
（解决方案描述）

## 评论
| 日期 | 评论人 | 评论 |
|------|--------|------|
| | | |
```

```
（1-2 句话描述）
```

## 使用建议

1. **模板选择**：根据文档类型选择对应模板
2. **占位符替换**：使用实际内容替换 {{ }} 中的占位符
3. **定制调整**：根据具体需求增删章节
4. **版本控制**：重要文档纳入版本控制
5. **定期回顾**：定期更新模板以适应变化的需求
