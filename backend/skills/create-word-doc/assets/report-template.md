# 标准报告模板

## 文档信息

| 字段 | 占位符 | 说明 |
|------|--------|------|
| 文档标题 | `{{ title }}` | 报告的主标题 |
| 报告编号 | `{{ report_no }}` | 内部报告编号 |
| 版本 | `{{ version }}` | 文档版本号 |
| 作者 | `{{ author }}` | 报告撰写人 |
| 部门 | `{{ department }}` | 所属部门 |
| 日期 | `{{ date }}` | 报告日期 |
| 保密级别 | `{{ confidentiality }}` | 公开/内部/机密 |
